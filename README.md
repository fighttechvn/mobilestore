# Mobile Store Listing Preview

A single-file, JSON-driven mockup of how an app looks on the **Apple App Store**
and **Google Play** store pages — useful for reviewing listing copy, screenshots
and graphics before you submit.

🔗 **Live demo:** https://fighttechvn.github.io/mobilestore/

> This deployment uses **dummy demo data** (a fictional "PulseFit" app). No real
> app credentials or identifiers are included.

## Features

- **Two store layouts** — toggle between App Store and Google Play renderings.
- **Localized** — switch locales; per-locale copy from one JSON file.
- **ASO checks** — live character-count limits per store field (name, subtitle,
  keywords, descriptions, changelog…).
- **Edit JSON** — edit the listing inline and hit *Apply & preview*; download the
  result as `listing.json`.
- **Load JSON…** — open any `listing.json` to preview your own app.
- **🖼️ Drag & drop images** — drop image files **or a whole folder** anywhere on
  the page (or use *Load images…*). They're auto-sorted into iPhone / iPad /
  phone / tablet galleries, app icon and feature graphic, by filename hints and
  aspect ratio. Phone shots mirror into both stores; tablet shots likewise.

## Use it with your own app

1. Open the live demo (or run locally — see below).
2. Click **Load JSON…** and pick your `listing.json` (same shape as the bundled
   `listing.json`).
3. Drag your screenshots (or their folder) onto the page to fill the galleries.
4. Tweak copy in **Edit JSON**, watch the **ASO checks**, then **Download
   listing.json**.

## Run locally

```bash
python3 -m http.server 8092
# open http://localhost:8092/
```

(Browsers block `fetch()` over `file://`, so it must be served over http.)

## `listing.json` shape

```jsonc
{
  "app":   { "name", "androidPackage", "iosBundleId", "versionName", "icon", "developer", "rating", "downloads", "locales": [...], ... },
  "limits": { "appstore": {...}, "googleplay": {...} },     // ASO character limits
  "locales": { "en-US": { "displayName", "appstore": {...}, "googleplay": {...}, "fullDescription", "whatsNew" }, ... },
  "screenshots": { "iphone": [{file,label,size}], "ipad": [...], "phone": [...], "tablet": [...] },
  "graphics": { "appIcon": {status,file,spec}, "featureGraphic": {status,file,spec} }
}
```

Generated by the `store-release` skill.
